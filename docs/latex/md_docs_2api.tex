Проект разделён на три слоя:\+


\begin{DoxyEnumerate}
\item {\ttfamily src/\+common} Здесь лежит общий line-\/delimited JSON протокол, сериализация сообщений и описание флота.
\item {\ttfamily src/\+server} {\ttfamily \doxylink{class_server_application}{Server\+Application}} принимает TCP-\/соединения и роутит команды. {\ttfamily \doxylink{class_game_manager}{Game\+Manager}} управляет комнатами, очередностью ходов и правилами партии. {\ttfamily \doxylink{class_game_board}{Game\+Board}} валидирует расстановку кораблей и обрабатывает выстрелы. {\ttfamily \doxylink{class_database_manager}{Database\+Manager}} реализован как singleton над SQLite и хранит пользователей и статистику.
\item {\ttfamily src/\+client} {\ttfamily \doxylink{class_api_client}{Api\+Client}} реализован как singleton и поддерживает одно TCP-\/соединение к серверу. {\ttfamily \doxylink{class_command_line_client}{Command\+Line\+Client}} даёт REPL и одноразовые команды для тестирования API без GUI.
\end{DoxyEnumerate}\hypertarget{md_docs_2api_autotoc_md2}{}\doxysection{\texorpdfstring{Транспорт}{Транспорт}}\label{md_docs_2api_autotoc_md2}
